# Hackathon Finder
A web app that allows you to search for available hackathons in different locations.

Check it out here : http://monsuratolaosebikan.github.io/hackathon-finder/

**API's**
* Google Maps (JavaScript) 
* Eventbrite

**Frameworks**
* Knockout.js
* Bootstrap
